# 函数: isMobile()

> **isMobile**(): `boolean`

判断当前环境是否为移动设备

## 返回值类型

`boolean`

返回一个bolean值，true表示为移动端，false为非移动端

## 示例

```ts
isMobile()   //  false当前环境为浏览器
```
