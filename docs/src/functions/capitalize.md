# 函数: capitalize()

> **capitalize**(`str`): `string`

将单词的首字母转为大写字母

## 参数

| 参数名 | 类型 | 描述 |
| :------ | :------ | :------ |
| `str` | `string` | 需要转化的英文字符串 |

## 返回值类型

`string`

返回一个以大写字母为开头的原单词

## 示例

```ts
console.log(capitalize(capitalize)) //   Capitalize
```
