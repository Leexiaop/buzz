# 函数: subtraction()

> **subtraction**(`num1`, `num2`): `string`

计算任意俩个正整数数的差。

## 参数

| 参数名 | 类型 | 描述 |
| :------ | :------ | :------ |
| `num1` | `string` | 第一个参数 |
| `num2` | `string` | 第二个参数 |

## 返回值类型

`string`

返回一个俩数相减的差

## 示例

```ts
subtraction('1', '1')    //  0
```

## Remarks

*当前方法不支持负数的相减
