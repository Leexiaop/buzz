# 函数: multiplication()

> **multiplication**(`a`, `b`): `string`

计算俩个正整数相乘

## 参数

| 参数名 | 类型 | 描述 |
| :------ | :------ | :------ |
| `a` | `string` |  |
| `b` | `string` |  |

## 返回值类型

`string`

返回相乘后的积字符串

## 示例

```ts
console.log(multiplication('2', '3'))    //  '6'
```

## Remark

当前方法不支持负数的相乘
