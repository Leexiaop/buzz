# 函数: add()

### 这是一个计算俩数相加的方法

> **add**(`a`, `b`): `number`

## 参数

| 参数名 | 类型 | 描述 |
| :------ | :------ | :------ |
| `a` | `number` | 要求是一个number, 必传 |
| `b` | `number` | 要求是一个number, 必传 |

## 返回值类型

`number`

(number) 返回俩个数相加后的结果

## 示例

```ts
console.log(add(3, 4)); // 7
```
