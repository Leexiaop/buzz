# 函数: add()

### 计算任意俩个数的和

> **add**(`num1`, `num2`): `number`

## 参数

| 参数名 | 类型 | 描述 |
| :------ | :------ | :------ |
| `num1` | `number` | 要求是一个number, 必传 |
| `num2` | `number` | 要求是一个number, 必传 |

## 返回值类型

`number`

(number) 返回俩个数相加后的结果

## 示例

```ts
console.log(add(3, 4)); // 7
```
