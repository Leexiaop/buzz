# 函数: add()

> **add**(`num1`, `num2`): `number`

计算任意俩个数字的和。

## 参数

| 参数名 | 类型 | 描述 |
| :------ | :------ | :------ |
| `num1` | `number` | 数 |
| `num2` | `number` | 数 |

## 返回值类型

`number`

返回一个俩数相加后的和

## 示例

```ts
add(1, 1)    //  2
add(-1, 1)    //  0
```
