# 函数: validateIdNumber()

> **validateIdNumber**(`value`): `boolean`

验证15或者18位身份证号码是否合格。

## 参数

| 参数名 | 类型 | 描述 |
| :------ | :------ | :------ |
| `value` | `string` | 身份证号码 |

## 返回值类型

`boolean`

返回一个布尔值，通过返回true

## 示例

```ts
validateIdNumber('420102199003078776')    //  true
```
