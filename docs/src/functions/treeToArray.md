# 函数: treeToArray()

> **treeToArray**(`treeObj`, `rootid`): `any`[]

将一个对象结构，或者是数组结构的树形结构扁平后，返回一个一维的数组

## 参数

| 参数名 | 类型 | 描述 |
| :------ | :------ | :------ |
| `treeObj` | `any` | 可以是一个数组或者是一个对象 |
| `rootid` | `any` | 根节点 |

## 返回值类型

`any`[]

返回一个扁平后的一维数组
